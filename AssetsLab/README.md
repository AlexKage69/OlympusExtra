# Assets

## Importing Assets
```
texture_packing_wheel pk -s PackMe -b OEAssets
deppth pk -s OEAssets -t OEAssets.pkg
```

Put the `OEAssets` and `OEAssets.pkg` into the Packages folder. 

Run modimporter as usual.
